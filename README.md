### [FoodMart](https://github.com/YoanAsghar/FoodMart)
**Full-Stack Restaurant / E-commerce System**

Started as an HTML template and evolved into a complete web application with a robust backend.

**Key Features:**
- JWT Authentication (Admin & User roles)
- Full CRUD for products with categories and image upload
- Persistent shopping cart
- Order management with email notifications
- Complete admin dashboard
- Well-documented REST API with Swagger

**Technologies:**
- **Backend:** ASP.NET Core · Entity Framework Core · MySQL · JWT · MailKit
- **Frontend:** HTML + CSS + JavaScript (Bootstrap + jQuery)
- **Others:** Docker + Docker Compose · Swagger UI
